# booking-bot

Telegram-бот на Node.js, построенный на библиотеке [Telegraf](https://telegraf.js.org/).

На данный момент это базовый каркас: бот отвечает на команду `/start` и эхом повторяет любой присланный текст. Дальше планируется логика бронирования.

## Возможности

- `/start` — бот отвечает приветствием.
- Любое текстовое сообщение — бот отвечает `Ты написал <текст>`.
- Корректное завершение работы по сигналам `SIGINT` / `SIGTERM`.

## Требования

- [Node.js](https://nodejs.org/) (рекомендуется LTS-версия)
- Токен Telegram-бота от [@BotFather](https://t.me/BotFather)

## Установка

```bash
git clone <git clone https://github.com/vaicett/booking-bot.git>
cd booking-bot
npm install
```

## Настройка

Создай файл `.env` в корне проекта и добавь токен бота:

```env
BOT_TOKEN=твой_токен_от_BotFather
```

Файл `.env` уже добавлен в `.gitignore`, поэтому токен не попадёт в репозиторий.

## Запуск

```bash
node bot.js
```

После запуска в консоли появится сообщение `Бот запущен`.

## Структура проекта

```
